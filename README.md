## 📖 Project Overview

Logistic Regression is one of the most important supervised learning algorithms used for classification tasks. Unlike linear regression, which predicts continuous values, logistic regression estimates the probability that an observation belongs to a particular class.

The notebook introduces the theoretical foundations of logistic regression while demonstrating how probability-based classification models can be interpreted and visualized.

---

## 🧠 What is Logistic Regression?

Logistic Regression predicts the probability of a binary outcome using the **Sigmoid (Logistic) Function**.

The predicted probability ranges from:

- **0** → Negative Class
- **1** → Positive Class

A decision threshold (commonly 0.5) is then used to assign a class label.
